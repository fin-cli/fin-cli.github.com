Version 0.3.0 is now available.

### Changelog

Commands:

- added `fp sql`
- improved `fp option`
- improved `fp plugin` and `fp theme`
- added `fp generate`
- added `fp core version`

Internals:

- added `--version` special parameter
- added `--blog` global parameter
- added bash completion script
- added pear installer
