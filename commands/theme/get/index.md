`wp theme get` - Get a theme

### OPTIONS

&lt;theme&gt;
: The theme to get.

[\--field=&lt;field&gt;]
: Instead of returning the whole theme, returns the value of a single field.

[\--format=&lt;format&gt;]
: Output list as table or JSON. Defaults to table.

### EXAMPLES

    wp theme get twentytwelve --format=json
