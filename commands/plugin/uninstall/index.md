`wp plugin uninstall` - Uninstall a plugin.

### OPTIONS

&lt;plugin&gt;...
: One or more plugins to uninstall.

[\--no-delete]
: If set, the plugin files will not be deleted. Only the uninstall procedure
will be run.

### EXAMPLES

    wp plugin uninstall hello
