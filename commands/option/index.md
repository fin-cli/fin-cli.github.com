`wp option` - Manage options.

### OPTIONS

[\--format=json]
: Encode/decode values as JSON.

### EXAMPLES

    wp option get siteurl

    wp option add my_option foobar

    wp option update my_option '{&quot;foo&quot;: &quot;bar&quot;}' --format=json

    wp option delete my_option
