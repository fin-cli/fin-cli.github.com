`wp db` - Perform basic database operations.

### OPTIONS

\--yes
: Answer yes to the confirmation message.

&lt;file&gt;
: The name of the export file. If omitted, it will be '{dbname}.sql'

&lt;SQL&gt;
: A SQL query.

### EXAMPLES

    # execute a query stored in a file
    wp db query &lt; debug.sql
