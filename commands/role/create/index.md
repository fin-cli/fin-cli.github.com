`wp role create` - Create a new role.

### OPTIONS

&lt;role-key&gt;
: The internal name of the role.

&lt;role-name&gt;
: The publicly visible name of the role.

### EXAMPLES

    wp role create approver Approver

    wp role create productadmin &quot;Product Administrator&quot;
